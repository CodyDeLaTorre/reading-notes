## Cloning

You can create a copy of an existing Git repository from a particular server by using the clone command with a repository’s URL:

$ git clone + url

## Saving from git to github

1. git add .
2. git commit -m
3. git push origin main

## Undoing Actions

Commit Mistakes
You can use the –amend command when you need to alter a commit message or forgot to add some files.

$ git commit --amend
